# Go 源码注解

学习`Go`已经有一段时间了，但是只停留在表面层面，并没有深入了解内部原理，所以只能知道程序结果，但是却不知道深层次的原因。最近在优化代码时，感觉到对`Go` 的理解不够深入，因此创建了这个项目，希望可以在阅读源码中拨开眼前的迷雾。

> 源码面前，了无秘密
>             --  侯捷

## 源码版本

本源码注解是基于 `1.9.4` 进行的注解

## 使用的标记

* `+-` 表示内容的关键点
* `+-?` 表示有疑问的地方，需要以后回顾的时候重新填写
